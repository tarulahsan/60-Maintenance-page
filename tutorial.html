Customization Tutorial

Project structure

Each template lives in its own folder under maintenance_pages/ and includes index.html, style.css, script.js, logo.svg, and favicon.svg.

Change the logo (SVG)

Replace logo.svg in the template folder with your custom SVG, or edit colors and text inside. Update the image source if you rename the file:

<img src="logo.svg" alt="Your brand">

Tip: Keep viewBox sizes around 128–256 and use a transparent background for best results.

Update the favicon (SVG)

Replace favicon.svg and ensure the <link rel="icon"> in index.html points to it:

<link rel="icon" type="image/svg+xml" href="favicon.svg">

Countdown timer

  • Set target by days: In script.js, edit DEFAULT_DAYS.
  • Reset programmatically: Call resetCountdown(14) from the console or use the Reset button.
  • Override via URL: Append ?days=10 or ?date=2025-12-31T23:59:59Z to the URL.
  • Persisted: The target time is stored in localStorage. Clear it to restart.
// script.js
const DEFAULT_DAYS = 14;
resetCountdown(7); // sets 7 days from now

Change background and animations

Open style.css and adjust theme variables or switch the variant class on the <body>.

:root {
  --bg: #0f1021;
  --primary: #6a11cb;
  --accent: #2575fc;
}
<body class="v-glass"> ... </body>
<body class="v-neon">  ... </body>
<body class="v-wave">  ... </body>

Change title, paragraphs, and icons

Edit content in index.html. Icons use a CDN such as Remix Icons or Boxicons. Change icon classes to swap visuals:

<i class="ri-tools-fill"></i>  <!-- Remix Icons -->
<i class="bx bxs-cog"></i>     <!-- Boxicons -->

Colors and typography

Colors are defined with CSS variables in style.css. Fonts are loaded from Google Fonts in the document <head>.

Make it your own

  • Swap social links and contact details.
  • Adjust border radius and shadows via --radius and --shadow.
  • Change component layout by editing the grid in .card-inner.

Troubleshooting

  • If timer doesn’t move, confirm your system clock and localStorage availability.
  • Reset the countdown by removing the browser’s localStorage key mp_target_<slug>.
